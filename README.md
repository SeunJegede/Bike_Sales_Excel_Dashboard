# 🚲 Bike Sales Dashboard — Microsoft Excel Project

A Microsoft Excel project covering full data preparation and dashboard creation to explore customer behaviour and bike purchasing trends.

---

## 📌 Overview

This project demonstrates an end-to-end analytics workflow built entirely in Microsoft Excel, from raw data cleaning through pivot analysis to an interactive dashboard. The dataset captures bike sales records and provides insight into the types of customers most likely to purchase a bike.

---

## 🚀 Workflow

1. **Data Acquisition** — Sourced a raw bike sales dataset for analysis
2. **Data Preservation** — Duplicated the raw sheet into a working copy to keep source data intact
3. **Data Cleaning** — Identified and resolved data quality issues across key columns
4. **Pivot Analysis** — Built PivotTables to answer targeted business questions
5. **Dashboard Build** — Assembled PivotCharts and slicers into an interactive summary view

---

## 🧹 Data Cleaning Steps

The raw dataset required several fixes before analysis could begin:

- Identified and removed **duplicate rows**
- Corrected inconsistent **data types and text formatting**
- Standardised values across key categorical columns
- Engineered **additional columns** to support richer analysis
